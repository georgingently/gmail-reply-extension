# Gmail Reply Extension

A Chrome Extension that reads email content, understands context, and uses Groq LLM to generate replies based on user input.

## Features
- Reads email content (body, replies, forwards).
- Context-aware dialogue box for user interaction.
- Integration with Groq LLM for generating replies.

## Setup
1. Clone the repository.
2. Install dependencies (if any).
3. Load unpacked extension in Chrome.
